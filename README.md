# Zappy---AI-Serveur-GUI
Zappy est un projet réalisé dans le cadre de ma 2nd année à Epitech où plusieurs équipes s'affrontent sur une carte remplie de ressources. Chaque joueur, piloté par une IA autonome, se déplace, se nourrit et récolte des pierres pour évoluer en niveau via un rituel d'élévation. L'équipe gagnante est la première à atteindre le niveau maximum.
